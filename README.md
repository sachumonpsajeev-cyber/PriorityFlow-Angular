# 🚀 PriorityFlow-Angular

A modern, high-performance Task Manager built with **Angular 17+** utilizing the latest **Signals** API for reactive state management. 

![Angular](https://img.shields.io/badge/angular-%23DD0031.svg?style=for-the-badge&logo=angular&logoColor=white)

![CSS3](https://img.shields.io/badge/css3-%231572B6.svg?style=for-the-badge&logo=css3&logoColor=white)

## ✨ Key Features

- **⚡ Angular Signals:** Leverages `signal()` and `computed()` for optimized, granular DOM updates.
- **🎨 Priority Badging:** Color-coded status system (High, Medium, Low) for better task visualization.
- **🔍 Smart Filtering:** Toggle between `All`, `Active`, and `Done` states instantly.
- **💾 Persistent Storage:** Integrated with `LocalStorage` so your data survives browser refreshes.
- **✏️ Inline Editing:** Double-click any task to rename it instantly.
- **📱 Responsive UI:** Clean, centered card layout designed for both desktop and mobile views.
